# v0.0.3

Released: 2026-05-06

- Added GitHub Actions CI for pushes to `main`, pull requests, and manual
  dispatch, running the same `uv run tox` validation suite used locally.
- Added and documented the release workflow, including release-note generation,
  package artifact validation, PyPI Trusted Publishing, and GitHub Release
  creation from `docs/releases`.
- Refined the README introduction, usage flow, and disclaimer so the project is
  positioned as a local developer compatibility layer for Codex-backed
  OpenAI-compatible clients.
- Made Codex auth and config file writes atomic and private by default with
  `0o600` permissions.
- Preserved `reasoning.summary` when forwarding requests to the Codex HTTP
  backend.
- Tightened compatibility internals around stored chat metadata, server text
  helpers, and timeout handling.
