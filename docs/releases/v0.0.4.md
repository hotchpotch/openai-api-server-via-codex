# v0.0.4

Released: 2026-05-06

- Documented `OPENAI_BASE_URL` and `OPENAI_API_KEY` usage in README examples so
  `openai-python` can be configured through standard client environment
  variables.
- Clarified that the local server accepts any incoming API key by default and
  that `--api-key` is required for server-side request authentication.
- Added Python 3.10 support, including a `tomli` fallback for TOML config
  loading.
- Switched local and CI validation to Python 3.10 with `py310`, `lint`, and
  `type` tox environments.
- Updated release version checks to run under the uv-managed Python
  environment.
